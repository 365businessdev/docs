# Feature Overview

This section provides an overview of the key features of 365 business Banking. Since this section lists features that are in development or have been recently released, some features may not be available depending on region or customer type.

The **General Availability** column indicates when the feature will be delivered within the specified month. The delivery date can be any day within that month. Released features show the complete date, including the day of release.

This checkmark (<i aria-hidden="true" class="fas fa-check"></i>) indicates which features are released for public preview or early access, as well as for public preview, early access, and general availability.
