# TimeStamp Method
Sets the timestamp of a ExceptionResponse object.

## Syntax
```javascript
	TimeStamp(string: Text)
```

## Parameters
### *string*
Type: Text<br/>
Timestamp as a string to be assigned to the 'timestamp' exception property.

## Return Value
### *None*
