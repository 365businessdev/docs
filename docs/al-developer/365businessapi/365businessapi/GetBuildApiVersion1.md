# GetBuildApiVersion Method
Returns the build version no. of the currently used API Version.

## Syntax
```javascript
    [Integer := ] GetBuildApiVersion()
```

## Parameters
### *None*

## Return Value
### *Text*
Type: Integer<br/>
The build version no. of the 365 business API metadata.
