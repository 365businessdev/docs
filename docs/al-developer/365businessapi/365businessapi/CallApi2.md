# CallApi Method
Sends a POST request to the API.

## Syntax
```javascript
    CallApi(serviceName: Text; request: Text)
```

## Parameters
### *serviceName*
Type: Text<br/>
Name of the API service (e.g. `PDF`) to consume.
### *request*
Type: Text<br/>
Request message to be sent to the API service.

## Return Value
### *None*
