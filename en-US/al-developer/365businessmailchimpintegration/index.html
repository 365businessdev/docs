365 business Mailchimp Integration

The 365 business Mailchimp Integration app implements an interface to Mailchimp bulk e-mail service to manage audience and audience members directly in Microsoft Dynamics 365 Business Central.

Interfaces

The 365 business Mailchimp Integration app is providing codeunits to allow integration into customer specific business processes.

Mailchimp Interface (Codeunit 5523720 - bdev.Mailchimp Interface)

The bdev.Mailchimp Interface codeunit object is providing communication functionalities with the Mailchimp API.

Methods

The following methods are available:

Procedure Description
CallApi() Call Mailchimp API using GET method.
CallApi() Call Mailchimp API with request body using POST method.
CallApi() Call Mailchimp API.
ApiResponse() Returns the API Response from previous call as JSON object.
ApiResponse() Writes the API Response from previous call as JSON object.

Mailchimp Interface Calls (Codeunit 5523722 - bdev.Mailchimp Interface Calls)

The bdev.Mailchimp Interface Calls codeunit object is providing actions to update or create audience in Mailchimp.

Methods

The following methods are available:

Procedure Description
GetMemberInformation() Retrieve Member Information from Mailchimp
GetMemberInformation() Retrieve Member Information from Mailchimp
GetAudienceMembers() Retrieve Audience Members from Mailchimp
GetAudienceMembers() Retrieve Audience Members from Mailchimp
GetAudienceLists() Retrieve Audience Lists from Mailchimp
CreateAudienceList() Create Audience List at Mailchimp
DeleteAudienceList() Delete Audience List at Mailchimp
UpdateAudienceMember() Update Member Information at Mailchimp
UpdateAudienceMemberTag() Create or Update tags for Audience Member
UpdateAudienceMemberTag() Create, Update or Delete tags for Audience Member
