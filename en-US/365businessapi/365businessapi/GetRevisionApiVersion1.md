# GetRevisionApiVersion Method
Returns the revision version no. of the currently used API Version.

## Syntax
```javascript
    [Integer := ] GetRevisionApiVersion()
```

## Parameters
### *None*

## Return Value
### *Text*
Type: Integer<br/>
The revision version no. of the 365 business API metadata.
