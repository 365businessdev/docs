# Validate Method
Validate and auto-complete a ship-to address.

## Syntax
```javascript
Validate(var shipToAddress: Record "Ship-To Address")
```

## Parameters
### *shipToAddress*
Type: Record "Ship-To Address"<br/>
"Ship-To Address" record to validate.

## Return Value
### *None*
