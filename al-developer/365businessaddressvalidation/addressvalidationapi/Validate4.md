# Validate Method
Validate and auto-complete a alternative address.

## Syntax
```javascript
Validate(var altAddress: Record "Alternative Address")
```

## Parameters
### *altAddress*
Type: Record "Alternative Address"<br/>
"Alternative Address" record to validate.

## Return Value
### *None*
